Spyce
=====

Spyce is a cloud deployment toolkit developed and maintained by RATES to facilitate the development and maintenance of Cloud Services. Spyce functionality is described herein to document the RGVFlood development and deployment process.

Core Spyce functions essential to all deployments are described below. Spyce functionality used specifically in each stage of RGVFlood development are described in subsequent sections in context.
