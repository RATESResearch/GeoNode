Goals & Objectives
==================

This report describes the Implementation of the RGVFlood User Interface (project deliverable 1.4.1.4).  As outlined in the REON.cc Pre-Development Plan (prior project deliverabale 1.4.1.3), specific objectives for this implementation include:

-   Enable Data Assimilation into the RGVFlood cyberinfrastructure, including GIS Layers and Maps
-   Implement an LRGV-focused User Interface which can be used and evaluated by expected users
-   Provide a platform that can evolve to include new potential applications
